# Task 1

CLI chatbot using **Ollama**.

## Files

- `chatbot.py` — command-line chatbot script.

## Requirements

- Python 3.10+
- Ollama installed and running
- Model pulled locally (default: `llama3.2:1b`)

## Run

From this folder:

```bash
python chatbot.py
```

Type `exit` or `quit` to stop.

## Ollama setup (if needed)

```bash
ollama serve
ollama pull llama3.2:1b
```
