Lab Assignment: Loops

Objective

  • Practice using for, foreach, while, and do...while loops.
  • Learn to control loop execution using counters and conditions.
  • Create interactive Windows Forms UI to input data and display loop results.
  • Understand when to use each type of loop in practical scenarios.

Lab Description

In this lab, you will build a C# Windows Forms application that demonstrates the four types of loops. The program has three main sections:

  1. Number Printer Section – print numbers using a for loop.
  2. Letter Display Section – display each character of a text using a foreach loop.
  3. Interactive Counter Section – use while and do...while loops for user input.

Steps

Step 1: Number Printer Section (Using for Loop)

In Form1, create:

  • A TextBox to enter a positive integer n.
  • A Button to start printing numbers.
  • A Label to display numbers from 1 to n.

Behavior:

  • Use a for loop to generate numbers from 1 to n and display them.
  • If n <= 0, display "Please enter a positive integer."

Expected Output Example:

  • Input: n = 5
  • Label displays: 1 2 3 4 5
  • Input: n = -2
  • Label displays: Please enter a positive integer.

Hint: To display the intermediate values in each iteration in the loop, you can use string concatenation. Otherwise, you will only see the last iteration's value because the loop happens so fast, that every previous iteration is overwritten by the next iteration. Therefor you need to add the previous value in the next iteration by concatenation.

Step 2: Letter Display Section (Using foreach Loop)

In Form1, create:

  • A TextBox for the user to enter a word or sentence.
  • A Button labeled "Show Letters".
  • A Label to display each letter.

Behavior:

  • Use a foreach loop to iterate over each character in the input string and display it.
  • Ignore empty input and show a message like "Please enter some text."

Expected Output Example:

  • Input: Hello
  • Label displays:
    H
    e
    l
    l
    o
  • Input: (empty)
  • Label displays: Please enter some text.

Step 3: Interactive Counter Section (Using while and do...while Loops)

In Form1, create:

  • A TextBox for user input of a number.
  • Two Buttons: one for while loop, one for do...while loop.
  • A Label to display output.

Behavior:

  • While Loop Button: repeatedly double the number until it exceeds 100 and display each value.
  • Do...While Loop Button: repeatedly subtract 5 from the number until it is less than or equal to 0 and display each value.

Expected Output Examples:

  • Input: 10, While Loop Button → Label displays: 10 20 40 80 160
  • Input: 12, Do...While Loop Button → Label displays: 12 7 2 -3

Requirements:

  • Use all four loop types in separate sections of the program.
  • Validate user input (must be numeric and positive where applicable).
  • Display loop output clearly in the UI.
  • Ensure the program runs without errors.
